# rentcar00_OPS Future Ideas — 2026-05-17

## 문서 역할
이 문서는 확정 작업이 아니라 차후 추가하고 싶은 기능 아이디어 보관용이다.
현재 구현 기준이나 active 작업 범위로 해석하지 않는다.

---

## 차후 추가 후보

### 1. 담당자 지정
- 예약/일정/차량 상태에 담당 직원을 지정한다.
- 담당자별 업무량과 진행 상태를 볼 수 있게 한다.

### 2. 탁송 호출
- API 연동 가능한 탁송 업체를 수배한다.
- 예약/일정 상세에서 탁송 요청, 상태 조회, 비용 기록을 연결한다.

### 3. 직원 상태 자동 업데이트
- 배차 이동, 반납 이동, 탁송, 휴식, 출근/퇴근 등 직원 상태를 자동 반영한다.
- 일정 액션과 직원 상태가 연동되도록 한다.

### 4. 자동 사진 업로드
- 배차/반납/차량 상태 사진을 자동 업로드한다.
- 예약/차량/일정 단위로 사진을 묶는다.

### 5. 자동 배반차 로그
- 배차/반납 이동 및 완료 이력을 자동 로그화한다.
- 작업자, 시간, 위치/상태, 관련 예약/차량을 기록한다.

### 6. 면허증 사진 저장
- 고객 면허증 사진을 예약과 연결해 저장한다.
- 개인정보/민감정보 보관 정책과 접근권한이 필요하다.

## 보류 이유
- 권한, 개인정보, 외부 API, 운영비, 기기 권한 이슈가 있어 바로 active 범위에 넣지 않는다.
- 관리자/직원관리/작업로그 기반이 먼저 잡힌 뒤 phase화한다.
