## 실기기 QA 포인트
1. 대기 차량 배차
   - 배차 버튼 클릭 시 보험/일반/장기 선택지만 뜨는지 확인
   - 선택 즉시 현재 시각 기준으로 차량 상태가 바뀌는지 확인
   - 상세 수정은 별도 수정 버튼에서만 하는지 확인
2. 세차
   - 세차 다이얼로그에서 외부세차/실내세차를 연속 토글할 수 있는지 확인
   - 항목 클릭 후 다이얼로그가 닫히지 않는지 확인
   - X 또는 바깥 터치로 닫히는지 확인
3. 예약 연결 일정
   - 예약에서 생성된 배차/반납 일정은 시간만 수정 가능한지 확인
   - 일정 시간 수정 시 예약 원장의 대여/반납 시간이 같이 바뀌는지 확인
   - 단독 일정은 기존처럼 전체 수정 가능한지 확인
4. 일정 카드 구분
   - 예약 연결 일정은 연한 파란색 카드로 보이는지 확인
   - 단독 일정은 기존 흰색 카드인지 확인
