# rentcar00_OPS 현황판 레이어 문서 인덱스 v1

## 1. 문서 목적
이 문서는 `rentcar00_OPS`의 **현황판 레이어 전용 문서 인덱스**다.

원칙:
- 이 문서는 현황판 레이어만 다룬다.
- 예약 레이어 문서는 여기서 다루지 않는다.
- 현황판 화면 복제와 시트1 기준 판단은 이 문서 묶음으로 본다.

## 2. 현황판 레이어 기준 문서
### 1순위. 현황판 방향 메모
- `projects/rentcar00_OPS/docs/rentcar00_OPS-status-board-layer-note-v1.md`
- 역할: 상단 `예약 / 현황판` 2레이어 구조와 현황판 분리 원칙

### 2순위. 현황판 복제 기준
- `projects/rentcar00_OPS/docs/rentcar00_OPS-status-board-clone-spec-v1.md`
- 역할: 기존 AppSheet 하단 5탭 복제 기준

### 3순위. 현황판 UI 매핑
- `projects/rentcar00_OPS/docs/rentcar00_OPS-status-board-sheet1-ui-mapping-v1.md`
- 역할: `차량현황 > 시트1` 컬럼과 화면 요소 매핑

### 4순위. 공통 규칙
- `projects/rentcar00_OPS/docs/rentcar00_OPS-spec.md`
- 역할: 전체 제품 범위와 상위 원칙
- `projects/rentcar00_OPS/docs/rentcar00_OPS-naming-mapping-rules-v1.md`
- 역할: 공통 네이밍 / 키 규칙
